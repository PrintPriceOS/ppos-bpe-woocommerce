- Banner and icon images should use the PrintPricePro brand colors.
- PNG format recommended for all images.
- SVG versions may be provided as source files but WordPress.org requires PNG.
- These files are NOT distributed with the plugin ZIP — they are uploaded separately to WordPress.org SVN `assets/` directory.
